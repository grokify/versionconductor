## [v0.1.0] - 2026-01-31

### Highlights

- Automated dependency PR management across multiple GitHub organizations
- Policy-based auto-review and merge of Renovate/Dependabot PRs
- Maintenance release creation with semantic version bumping

### Added

- CLI with `scan`, `review`, `merge`, `release`, and `version` commands using Cobra framework ([`1c1e14b`](https://github.com/grokify/versionconductor/commit/1c1e14b))
- GitHub collector for dependency PR discovery across organizations ([`1c1e14b`](https://github.com/grokify/versionconductor/commit/1c1e14b))
- Detection of Renovate and Dependabot PRs by author identification ([`1c1e14b`](https://github.com/grokify/versionconductor/commit/1c1e14b))
- PR metadata extraction including dependency name, versions, and update type ([`1c1e14b`](https://github.com/grokify/versionconductor/commit/1c1e14b))
