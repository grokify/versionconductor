## [v0.1.0] - 2026-01-31

### Highlights

- Automated dependency PR management across multiple GitHub organizations
- Policy-based auto-review and merge of Renovate/Dependabot PRs
- Maintenance release creation with semantic version bumping

### Added

- CLI with `scan`, `review`, `merge`, `release`, and `version` commands using Cobra framework
- GitHub collector for dependency PR discovery across organizations
- Detection of Renovate and Dependabot PRs by author identification
- PR metadata extraction including dependency name, versions, and update type
- Cedar policy engine for auto-merge decisions
- Three built-in merge profiles: `aggressive`, `balanced`, `conservative`
- PR merger with squash, merge, and rebase strategies
- Release creator with semantic version bumping (patch releases)
- JSON and Markdown report formats
- Dry-run mode for all write operations

### Dependencies

- `github.com/google/go-github/v82` for GitHub API
- `github.com/grokify/gogithub` v0.7.0 for GitHub API helpers
- `github.com/spf13/cobra` v1.10.2 for CLI framework
- `github.com/spf13/viper` v1.21.0 for configuration management

### Documentation

- README with installation, usage, and examples
- Example Cedar policies for auto-merge scenarios
- Profile configuration examples
